# Legacy Static Files (Backup)

この`legacy_static_backup`ディレクトリには、旧来の静的HTML/CSS/JSファイルが保存されています。

## 内容
- `Title/`: 旧タイトル画面の静的ファイル
- `Setting/`: 旧設定画面の静的ファイル

## 注意
- これらのファイルは新しいVite+Reactベースの`/client`アプリに統合されました。
- 現在のプロジェクトでは使用されていません。
- デザイン・機能は`/client/src/title/`, `/client/src/setting/`に移行済みです。
- 必要に応じて参照用に保管されています。
